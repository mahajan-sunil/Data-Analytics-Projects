# HR Database Management System

## Project Overview
This project is based on an HR (Human Resource) Database Management System developed using SQL Server.  
The database is designed to manage employee records, departments, jobs, salaries, locations, and dependents.

The project demonstrates SQL concepts ranging from basic queries to advanced database operations used in real-world HR systems.

---

## Database Tables
The project includes the following tables:

- Regions
- Countries
- Locations
- Jobs
- Departments
- Employees
- Dependents
