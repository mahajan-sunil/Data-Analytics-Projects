# Sales Analysis Dashboard

## Project Overview
This project is an interactive Sales Analysis Dashboard created using Power BI. The dashboard helps analyze sales performance, profit trends, customer behavior, and regional insights using a Superstore sales dataset.

## Tools & Technologies Used
- Power BI
- Microsoft Excel
- DAX
- Data Visualization

## Dataset Information
Dataset Name:
- Sales_Analysis_Datasets.xlsx

The dataset contains:
- Sales data
- Profit information
- Customer details
- Product categories
- Regional sales performance
- Order trends

## Dashboard Features
- Interactive dashboard visuals
- KPI cards for Sales and Profit
- Sales trend analysis
- Category-wise performance
- Region-wise analysis
- Filters and slicers
- Business insights visualization

## Key Insights
- Identified top-performing categories
- Analyzed profit and loss trends
- Compared regional sales performance
- Tracked sales growth patterns
- Improved business decision-making through visualization

## Project Objectives
- Convert raw sales data into meaningful insights
- Build an interactive Power BI dashboard
- Practice data cleaning and visualization
- Improve business analytics understanding
